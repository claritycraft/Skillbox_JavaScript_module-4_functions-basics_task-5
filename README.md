### En:
#### Task 5
Create a function that takes the following parameters:
- the price of an item from an online store
- the quantity of that item in the cart
- the discount percentage
…and returns the total cost of all items after applying the discount.

#### Example
The program has the following values:
Item price — 25000
Quantity — 3
Discount — 20
Call: totalSum(priceItem, amountItem, discountItem)
Output: 60000

____________________________________________________________________________________________________________________

### Ru:
#### Задача 5
Создайте функцию, которая принимала бы в качестве параметров цену товара из интернет-магазина, количество товара в корзине и значение скидки в процентах, а возвращала бы итоговую стоимость всего количества товара со скидкой.

#### Пример:

В программе записана цена товара — 25000, количество товара — 3, скидка — 20.

Вызов: totalSum(priceItem, amountItem, discountItem)

Вывод: 60000
